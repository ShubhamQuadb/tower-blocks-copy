## Controls

### Numeric Keypad
- **2**: Move UP
- **8**: Move DOWN
- **4**: Move LEFT
- **6**: Move RIGHT
- **5** or **Enter**: SHOOT / SELECT

### D-Pad Navigation
- **Arrow Keys**: Move spaceship
- **Center Button**: Shoot / Select

## Installation

### For KaiOS Simulator
1. Open Firefox WebIDE
2. Install KaiOS Simulator
3. Open Packaged App
4. Select this folder
5. Click Play

### For JioPhone/KaiOS Device
1. Enable Developer Mode
2. Connect via USB
3. Use Firefox WebIDE
4. Select USB device
5. Install and run

## Requirements
- KaiOS 2.0+ device
- 240x320 minimum screen
- JavaScript-enabled browser
- 512MB+ RAM recommended
