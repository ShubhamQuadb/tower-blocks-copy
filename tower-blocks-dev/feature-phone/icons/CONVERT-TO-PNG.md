# Icon Conversion Guide

SVG icons created! KaiOS simulator me test karne ke liye PNG format better hai.

## Quick Online Conversion:

### Method 1: CloudConvert (Recommended)
1. Go to: https://cloudconvert.com/svg-to-png
2. Upload `icon-56.svg` and `icon-112.svg`
3. Convert to PNG
4. Download and replace SVG files

### Method 2: Photopea (Free Photoshop)
1. Go to: https://www.photopea.com/
2. Open SVG file
3. Export as PNG
4. Save in this folder

### Method 3: Any Image Editor
- GIMP (Free): https://www.gimp.org/
- Paint.NET (Windows): https://www.getpaint.net/
- Preview (Mac): Built-in

## Icon Specs:
- **icon-56.png**: 56x56 pixels
- **icon-112.png**: 112x112 pixels
- **Format**: PNG with transparency
- **Design**: Purple gradient with white stacked blocks

## After Conversion:
1. Replace .svg files with .png files
2. Update manifest.webapp to use .png instead of .svg
3. Test in KaiOS simulator

---

Icons are ready to use as SVG, but PNG recommended for better compatibility.
