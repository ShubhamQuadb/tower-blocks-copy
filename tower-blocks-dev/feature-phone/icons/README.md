# App Icons for KaiOS

Place your app icons in this folder:

## Required Icons:

1. **icon-56.png** - 56x56 pixels
2. **icon-112.png** - 112x112 pixels

## Icon Guidelines:

- Use PNG format with transparency
- Keep design simple and recognizable
- Use bright colors for visibility
- Center your design
- No text (logo/symbol only)

## Quick Icon Creation:

You can use online tools like:
- https://favicon.io/
- https://realfavicongenerator.net/
- Canva
- Adobe Express

Or use image editing software like:
- GIMP (Free)
- Photoshop
- Paint.NET

## Icon Specifications:

```
icon-56.png:
- Size: 56x56 pixels
- Format: PNG
- Transparency: Recommended
- Used for: App launcher

icon-112.png:
- Size: 112x112 pixels  
- Format: PNG
- Transparency: Recommended
- Used for: High-DPI displays
```

## Example Design Ideas:

- Stack of colorful blocks
- Tower icon
- Letter "T" with blocks
- Simple geometric tower shape

---

**Note**: The app will work without custom icons, but KaiOS will use a default placeholder icon.
