# Tower Blocks Icons

This folder contains the app icons for Tower Blocks smartphone version.

## Files:
- `icon-112.svg` - 112x112px icon
- `icon-56.svg` - 56x56px icon

## Design:
- Purple gradient background (#667eea to #764ba2)
- White stacked blocks representing the tower blocks game
- Rounded corners for modern look

## Usage:
These SVG icons can be:
1. Used directly in web apps
2. Converted to PNG for app icons
3. Scaled to any size needed

## To Convert to PNG:
Use any SVG to PNG converter or image editing software like:
- Online: svg2png.com, cloudconvert.com
- Desktop: Inkscape, GIMP, Adobe Illustrator
